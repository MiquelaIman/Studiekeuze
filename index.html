Studiekeuze

Studiekeuze is een lastige keuze. Er is heel veel keuze en het is een keuze die belangrijk is voor de rest van je leven. Met deze website willen we jonge mensen helpen met hun keuzen. Succes! Hedda en Miquela.

WO

Je kan bij twee universiteiten een universitaire studie doen:

  • Vrije Universiteit Website van VU
  • Universiteit van Amsterdam Website van UvA

Deze universiteiten bieden een breed scala aan studies aan, op ene hoog niveau.

HBO

Je kan op verschillende plekken ene HBO opleinding door in Amsterdam, maar de twee bekendste plekken zijn:

  • Hogeschool van Amsterdam Website van HvA
  • InHolland Website van InHolland

Op een hogeschool leer je een beroep maar op een hoogniveau. Om naar een hogeschool te kunnen moet je een havo diploma hebben of een MBO opleiding hebben afgerond.

MBO

In Amsterdam kan je bij het ROC een opleiding doen

  • ROC Website van ROC Amsterdam

Bij een MBO opleiding leer je een beroep. Om een opleiding te kunnen doen moet je een vmbo diploma of hoger hebben. Er zijn heel veel verschillende MBO opleidingen.
